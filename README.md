# m-fanduel-notes-c1ed

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对任何单一域名或特定网站。

## 项目简介

这是一个以归档为目的的仓库，主要存放一些零散的 HTML 页面资源。页面内容各自独立，彼此之间没有关联。仓库本身不提供动态服务，仅作为静态页面的存储与展示空间。

## 目录说明

```
m-fanduel-notes-c1ed/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── README.md           # 本文件
```

- **index.html**：可选的主页，用于快速导航到各页面。
- **pages/**：所有独立 HTML 页面均放置在此目录下，文件名以简单数字或描述性名称命名。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不含外部依赖（除非在页面内显式引用 CDN 资源）。
- 页面内容可能涉及不同主题，但均不指向任何具体网站或服务。
- 归档仅用于保存与查阅，不保证页面内容的时效性或完整性。

## 维护说明

- 本仓库不定期更新，新增的页面会直接放入 `pages/` 目录。
- 如需修改或删除页面，请直接操作对应文件。
- 欢迎提交 Issue 或 Pull Request，但请勿添加与归档目的无关的内容。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/m-fanduel-notes-c1ed.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
